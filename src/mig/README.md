# MIG (Mach Interface Generator) stubs

These files are generated by
[MIG](https://developer.apple.com/library/archive/documentation/Darwin/Conceptual/KernelProgramming/Mach/Mach.html),
the Mach Interface Generator, from the `.defs` files shipped with macOS.

## Re-generating

On a macOS host with Xcode (or the Command Line Tools) installed:

```sh
mig $SDKROOT/usr/include/mach/mach_exc.defs
mig $SDKROOT/usr/include/mach/exc.defs
```

and add the Apple / Mach Operating System licenses to the top of each file.
